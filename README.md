# Tripigo

Welcome to Tripigo! An intelligent travel planning application designed to help you create the perfect trip.

![Tripigo Logo](tripigo%20logo.png)

## 🌍 About Tripigo

Tripigo is a modern travel planning application that leverages artificial intelligence to provide personalized travel recommendations. Whether you're planning a weekend getaway or a month-long adventure, Tripigo makes it easy to discover destinations, plan itineraries, and book accommodations all in one place.

## ✨ Key Features

### 🤖 AI-Powered Recommendations
- Intelligent destination suggestions based on your travel style and preferences
- Smart activity recommendations tailored to your interests
- Accommodation suggestions filtered by budget, location, and amenities
- Real-time weather insights and best time to visit information
- Personalized dining and entertainment recommendations

### 🗺️ Interactive Maps
- Explore destinations with integrated Leaflet mapping technology
- View attractions, hotels, and restaurants on interactive maps
- Get directions and distances between locations
- Visualize your itinerary geographically
- Mark favorite spots and save locations

### 📅 Itinerary Planning
- Create multi-day travel itineraries with ease
- Drag-and-drop activity organization
- Time-based scheduling and duration planning
- Add notes, photos, and links to each activity
- Share itineraries with travel companions
- Export itineraries as PDFs or shareable links

### 🔐 User Authentication
- Secure user registration and login
- Social authentication support (Google, GitHub, etc.)
- Personal travel profiles and preferences
- Save multiple itineraries and trip plans
- Access your travels from any device

### 📱 Responsive Design
- Works seamlessly on desktop, tablet, and mobile devices
- Progressive web app functionality
- Offline access to saved itineraries
- Touch-optimized interface for mobile users
- Fast loading times and smooth performance

### 💰 Budget Management
- Set trip budgets and track expenses
- Categorize spending (accommodation, food, activities, transport)
- Real-time expense calculations
- Currency conversion tools
- Budget alerts and notifications

### 🎫 Booking Integration
- Compare prices across multiple accommodation providers
- One-click booking for hotels and activities
- Flight search and booking assistance
- Transportation options (flights, trains, buses, car rentals)
- Reviews and ratings from verified travelers

### 👥 Social Features
- Invite friends to collaborate on trip planning
- Share itineraries and get feedback
- Comment and add suggestions to trips
- Build a travel wishlist and save inspiration
- Connect with fellow travelers

### 🌐 Multi-Language Support
- Support for multiple languages
- Currency conversion and localization
- Regional preferences and time zones
- Translated attraction descriptions and recommendations

## 🛠️ Technology Stack

### Frontend
- **React 18** - Modern UI library for building interactive interfaces
- **TypeScript** - Type-safe JavaScript for better code quality
- **Vite** - Lightning-fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **shadcn/ui** - High-quality React components built on Radix UI
- **React Router** - Client-side routing for seamless navigation

### Backend & Database
- **Supabase** - Open-source Firebase alternative for authentication and database
- **PostgreSQL** - Robust relational database (via Supabase)

### Maps & Location
- **Leaflet** - Interactive mapping library for destination exploration
- **Geolocation API** - User location services

### Forms & State Management
- **React Hook Form** - Efficient form handling with minimal re-renders
- **TanStack Query** - Powerful data fetching, caching, and synchronization
- **Zod** - TypeScript-first schema validation

### UI & Visualization
- **Recharts** - Composable charting library for budget visualization
- **Lucide React** - Beautiful, consistent icon library
- **Sonner** - Toast notifications for user feedback

### Utilities
- **Date-fns** - Modern date utility library for date handling
- **Tailwind Merge** - Utility to merge Tailwind CSS classes intelligently
- **React Markdown** - Markdown rendering for rich content
